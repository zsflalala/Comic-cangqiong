# comic-cangqiong

利用python异步协程，爬取斗破苍穹漫画，截至410话怨灵
使用的技术有异步请求，异步持久化存储，（异步文件存储）,爬虫基本图片存储
有线程池+异步协程加快下载的想法,但没有验证是否可行

## Dependencies

Python   3.7
asyncio  1.5.5 
aiohttp  3.8.3 
aiofiles 22.1.0
